# 使用vuejs搭建的一个单页面应用售卖平台项目实例，
涉及技术栈：vuex、vue-router、vue-resource、less、webpack 


## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
